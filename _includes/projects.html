Projects

Investigation into Energy Efficiency of Vector Field Based CNC Toolpaths - Compiled, modified, and integrated C++ software for the generation of vector-field based toolpaths for CNC machining. Improved process efficiency of previous approach by over 6.5%. Discovered systematic errors in previously published work and implemented improvements to experiments in order to make data collection more meaningful.
3D Printer Word Search Solver - Used optical character recognition to solve word search puzzles. Code generates g-code for a 3D printer such that it can solve the puzzles with an attached writing instrument. Designed a suitable writing instrument attachment for my 3D printer's toolhead.
Camera Interact - UC Berkeley Computer Graphics final project: An interactive lesson aimed at teaching camera concepts written in THREE.js
Embedded Sensors in Additively Manufactured Parts - UC Davis Mechanical Engineering senior project exploring the feasibility and accuracy of embedded sensors in 3D printed parts. Compared experimental beam strains to CAD simulations. Created Wheatstone bridge circuit to measure internal strains in 3D printed beams using embedded strain gauges
Wanhao 3D Printer Rebuild - Upgraded printer electronics using an Arduino Mega and a RAMPS shield. Changed wire type to Dupont connectors. Routed heated bed power through a MOSFET. Performed a Z-brace modification for system stiffness
